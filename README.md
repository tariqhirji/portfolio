A Personal Portfolio I built for myself to keep track of all my projects.
It will be maintained and updated throughout the years with cool new features as I develop them.

Future Iteration Improvements:

- Project routing
- In/Out of form skill bars
- Hero with me in front of blob

This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app).

## Available Scripts

First run 'npm install' to install the node_modules required for this project.
Then, in the project directory, you can run:

### `npm start`

Which runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

The page will reload when you make changes.\
You may also see any lint errors in the console.
